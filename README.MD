# YouTube Search

**youtube-search** is a command-line application that allows users to search for YouTube videos, display the results in a user-friendly list, and play the selected video using **mpv**. This tool uses the YouTube's search API to fetch video data and presents it in a terminal-based user interface.

## Dependencies

- `gcc` c compiler
- `libcurl` library for making HTTP requests
- `cJSON` library for JSON parsing
- `mpv` installed on your system for video playback

## Compiling

  **Clone this repository and install dependencies**:
   ```bash
   git clone https://github.com/YourUsername/youtube-search.git
   cd youtube-search
   make```
